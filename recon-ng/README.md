# Recon-ng

## Overview

Recon-ng is a modular Open Source Intelligence (OSINT) framework designed to automate the reconnaissance phase of a security assessment. It provides a workspace-based environment where different reconnaissance modules can be installed and used to collect publicly available information about a target.

For this project, Recon-ng was the first tool used to begin the passive reconnaissance of **OWASP**. The goal was to explore its capabilities, learn how its modular architecture works, and gather publicly available information without interacting directly with the target's infrastructure.

---

## Objectives

The objectives of using Recon-ng were to:

- Install and configure the Recon-ng framework.
- Explore the Marketplace and understand how modules are managed.
- Learn how API-based modules work.
- Perform passive reconnaissance against OWASP.
- Document the setup process, challenges encountered, and lessons learned.
