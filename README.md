#  Digital Shadow Assessment


## Overview

This repository documents a **Digital Shadow Assessment** conducted against the **OWASP (Open Worldwide Application Security Project)** as part of my learning journey in the **ALX Cybersecurity Programme**.

The objective of this project was to perform **passive Open Source Intelligence (OSINT)** reconnaissance to understand how publicly available information can reveal an organisation's digital footprint.

Throughout this assessment, I explored industry-recognised reconnaissance tools, documented my workflow, recorded challenges, and analysed the findings gathered from publicly accessible sources.

This repository is more than a final report—it's a complete record of my investigation, including the lessons learned, troubleshooting process, reference notes, and command cheat sheets.

---

## Objectives

- Learn passive reconnaissance techniques.
- Investigate the public digital footprint of OWASP.
- Gain hands-on experience with professional OSINT tools.
- Compare the strengths and limitations of different reconnaissance tools.
- Document the complete investigation process.
